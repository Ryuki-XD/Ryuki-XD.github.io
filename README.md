# Sudip Kr. Gachhadar — Developer Portfolio

Personal portfolio site, live at **https://ryuki-xd.github.io**.

Showcases my projects — an AI-powered business website, an AI resume analyzer, and a
JavaFX library management system — along with skills, services, and contact details.

## Tech stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui components

## Development

```sh
npm install
npm run dev      # dev server on http://localhost:8080
npm run build    # production build to dist/
```

## Deployment

Pushed to `Ryuki-XD/Ryuki-XD.github.io`, where a GitHub Actions workflow builds the site
and deploys it to GitHub Pages automatically.
